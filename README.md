# MoodDiaryEvoTest
Тестовое задание для EVO Mobile

## Неопределенности
В процессе разработки возникли вопросы по реализации некоторых аспектов приложения, по комментарию из макета Figma "Если что-то непонятно - можете сделать как считаете нужным.", я решил их по своему:
- Чтобы сменить режим отображения календаря необходимо нажать на год или месяц
- Поле заметок всегда активно для ввода
- Дата и время обновляется каждую минуту
- Все даты используют системную локализацию
- При нажатии на "Сегодня" автоматический скролл до текущего месяца
